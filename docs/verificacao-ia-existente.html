📌 Objetivo

Este documento é usado pelo CoE Operacional para auditar sistemas de IA que estão em uso mas foram criados sem passar pelo funil do CoE. O resultado determina se o sistema pode continuar operando, precisa ser adaptado aos padrões ou deve ser desativado.
